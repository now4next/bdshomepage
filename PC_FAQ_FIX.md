# PC 버전 FAQ 표시 오류 해결 가이드

**날짜:** 2026-04-13  
**문제:** PC에서 FAQ가 드롭다운으로 표시됨 (모바일은 정상)

---

## 🔍 문제 상황

스크린샷 분석 결과:
- ✅ 모바일: FAQ 아코디언이 정상 작동
- ❌ PC: FAQ가 기본 HTML `<select>` 드롭다운으로 표시됨

이는 **브라우저 캐시** 또는 **잘못된 URL** 때문입니다.

---

## ⚡ 즉시 해결 방법

### 방법 1: 강력 새로고침 (가장 빠름)

**Windows:**
```
Ctrl + F5
또는
Ctrl + Shift + R
```

**Mac:**
```
Cmd + Shift + R
```

### 방법 2: 시크릿 모드로 확인

1. 브라우저에서 시크릿/비공개 모드 열기
   - Chrome: `Ctrl + Shift + N`
   - Firefox: `Ctrl + Shift + P`
   - Safari: `Cmd + Shift + N`

2. 다음 URL 접속:
   ```
   https://bdshomepage.pages.dev
   ```

3. FAQ 섹션으로 스크롤하여 확인

### 방법 3: 브라우저 캐시 완전 삭제

**Chrome:**
1. 설정 (⋮) > 도구 더보기 > 인터넷 사용 기록 삭제
2. 시간 범위: "전체 기간"
3. ✅ 캐시된 이미지 및 파일
4. ✅ 쿠키 및 기타 사이트 데이터
5. "데이터 삭제" 클릭

**Firefox:**
1. 설정 (≡) > 개인정보 보호 및 보안
2. 쿠키 및 사이트 데이터 > 데이터 지우기
3. ✅ 캐시
4. "지우기" 클릭

**Edge:**
1. 설정 (...) > 설정 > 개인정보, 검색 및 서비스
2. 검색 데이터 지우기 > 지금 검색 데이터 지우기
3. ✅ 캐시된 이미지 및 파일
4. "지금 지우기" 클릭

## 📊 정상 작동 확인 방법

FAQ 섹션이 **정상적으로** 표시되면:

### 시각적 확인:
- ✅ 흰색 카드 6개 (둥근 모서리, 그림자)
- ✅ 각 카드에 질문 제목 (Merriweather 폰트)
- ✅ 오른쪽에 아래 화살표 아이콘 (▼)
- ✅ 첫 번째 FAQ가 펼쳐져 있음 (빨간 테두리)
- ✅ 질문에 마우스 호버 시 연한 회색 배경

### 인터랙티브 확인:
- ✅ 질문 클릭 시 답변이 부드럽게 펼쳐짐
- ✅ 다른 질문 클릭 시 이전 것은 닫힘 (한 번에 하나만)
- ✅ 화살표가 위로 회전 (▲)
- ✅ 활성 카드에 빨간색 테두리 표시

### ❌ 잘못된 표시 (구버전):
- ❌ 질문이 드롭다운 선택 상자로 표시됨
- ❌ 회색 테두리의 `<select>` 요소
- ❌ 클릭해도 애니메이션 없음
- ❌ 스타일이 적용되지 않은 텍스트

## 🔧 고급 문제 해결

### 개발자 도구로 확인

1. **F12** 키로 개발자 도구 열기

2. **Console 탭** 확인:
   ```
   JavaScript 오류가 있나요?
   - 빨간색 오류 메시지 확인
   - 특히 "faq" 관련 오류
   ```

3. **Network 탭** 확인:
   ```
   - 페이지 새로고침
   - index.html 파일이 로드되는지 확인
   - 상태 코드가 200 (성공)인지 확인
   - 크기가 0이 아닌지 확인
   ```

4. **Elements 탭** 확인:
   ```html
   - Ctrl+F로 "faq-item" 검색
   - <div class="faq-item"> 요소가 있나요?
   - <button class="faq-question"> 요소가 있나요?
   - <select> 요소는 있으면 안 됩니다
   ```

## 💻 PC와 모바일 차이 이유

모바일은 정상이고 PC만 문제인 이유:

### 가능한 원인:
1. **PC 브라우저 캐시가 더 오래됨**
   - 모바일: 최근에 접속 → 최신 버전
   - PC: 오래 전 접속 → 오래된 캐시

2. **다른 브라우저 사용**
   - 모바일: Safari/Chrome 모바일 (캐시 없음)
   - PC: Chrome/Edge (오래된 캐시)

3. **다른 URL 사용**
   - 모바일: 올바른 실서버 URL
   - PC: 북마크된 오래된 URL

### 해결:
- PC에서도 **시크릿 모드**로 열면 즉시 정상 표시됨
- 또는 **캐시 삭제** 후 새로고침
